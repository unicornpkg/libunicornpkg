# `io.sc3.p`

`io.sc3.p` is the identifier for the package provider of [SCPaste](https://p.sc3.io/).

This provider enables sourcing one or more pastes from SCPaste.

## `instdat.filemaps`

A table which contains mappings for pastes on SCPaste and the local file they will be placed in.

The key for each item in the table should correspond to the paste ID on SCPaste, and the value should be the local destination.
